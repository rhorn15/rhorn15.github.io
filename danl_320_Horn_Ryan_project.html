DANL 320 Final Project: Do Early Injuries Hurt Team Success?

Author

Ryan Horn

1. Introduction

This project asks a simple question:

If a team has more injury problems early in the season, does that team usually finish with fewer wins?

I use NFL injury reports from Weeks 1-8 and compare them to end-of-season results.

2. Literature Review (Optional)

A full literature review is optional for this project, so I focus on data, modeling, and clear interpretation.

3. Data

3.1 Source and Scope

I use only nflreadr data:

  • load_injuries() for injury and practice report records
  • load_schedules() for final game results

load_injuries_local <- function(path = "data/injuries.rds", refresh = FALSE) {
  if (!file.exists(path) || isTRUE(refresh)) {
    x <- nflreadr::load_injuries(seasons = TRUE)
    saveRDS(x, path)
  }
  readRDS(path)
}

load_schedules_local <- function(path = "data/schedules.rds", refresh = FALSE) {

3.2 Team Outcomes (What We Predict)

I build team-season outcomes from schedule results:

  • final regular-season win_pct
  • playoff qualification (Yes / No)
Code 
reg_games <- schedules_raw |>
  filter(season >= 2009, game_type == "REG", !is.na(home_score), !is.na(away_score))

reg_long <- bind_rows(
  reg_games |>
    transmute(
      season,
      team = home_team,
      points_for = home_score,
      points_allowed = away_score,
      win = as.integer(home_score > away_score),
      tie = as.integer(home_score == away_score)
    ),
  reg_games |>
    transmute(
      season,
      team = away_team,
      points_for = away_score,
      points_allowed = home_score,
      win = as.integer(away_score > home_score),
      tie = as.integer(away_score == home_score)
    )
)

team_outcomes <- reg_long |>
  group_by(season, team) |>
  summarise(
    games = n(),
    wins = sum(win),
    ties = sum(tie),
    win_pct = (wins + 0.5 * ties) / games,
    point_diff_pg = (sum(points_for) - sum(points_allowed)) / games,
    .groups = "drop"
  )

post_teams <- schedules_raw |>
  filter(season >= 2009, game_type != "REG") |>
  transmute(season, team = home_team) |>
  bind_rows(
    schedules_raw |>
      filter(season >= 2009, game_type != "REG") |>
      transmute(season, team = away_team)
  ) |>
  distinct()

team_outcomes <- team_outcomes |>
  left_join(post_teams |>
              mutate(playoff_flag = TRUE),
            by = c("season", "team")) |>
  mutate(
    playoff_flag = replace_na(playoff_flag, FALSE),
    playoff_label = factor(if_else(playoff_flag, "Yes", "No"), levels = c("No", "Yes"))
  )

team_outcomes |>
  summarise(
    team_seasons = n(),
    avg_win_pct = mean(win_pct),
    playoff_rate = mean(playoff_flag)
  )

3.3 Injury Features (Weeks 1-8 Only)

To keep this realistic, I only use injury information from the first half of the season.

I intentionally keep the features simple:

  • injury_reports: total injury-report entries
  • out_reports: players listed as Out
  • dnp_practice: did not practice or definitely out
  • injury_severity_score: weighted burden score
  • offense_report_share: share of injury reports on offense
  • defense_report_share: share of injury reports on defense
Code 
injuries <- injuries_raw |>
  filter(season >= 2009, game_type == "REG", week <= 8) |>
  mutate(
    report_status = str_squish(report_status),
    practice_status = str_squish(practice_status),
    position = str_squish(position),
    report_status = na_if(report_status, ""),
    practice_status = na_if(practice_status, ""),
    is_offense = position %in% c("QB", "RB", "FB", "WR", "TE", "T", "G", "C"),
    is_defense = position %in% c("DE", "DT", "LB", "CB", "S")
  )

injury_features <- injuries |>
  group_by(season, team) |>
  summarise(
    injury_reports = n(),
    out_reports = sum(report_status == "Out", na.rm = TRUE),
    dnp_practice = sum(practice_status %in% c("Did Not Participate In Practice", "Out (Definitely Will Not Play)"), na.rm = TRUE),
    injury_severity_score =
      3 * sum(report_status == "Out", na.rm = TRUE) +
      2 * sum(report_status == "Doubtful", na.rm = TRUE) +
      1 * sum(report_status == "Questionable", na.rm = TRUE) +
      0.5 * sum(report_status == "Probable", na.rm = TRUE),
    offense_report_share = mean(is_offense, na.rm = TRUE),
    defense_report_share = mean(is_defense, na.rm = TRUE),
    .groups = "drop"
  )

model_df <- team_outcomes |>
  left_join(injury_features, by = c("season", "team"))

feature_cols <- c(
  "injury_reports",
  "out_reports",
  "dnp_practice",
  "injury_severity_score",
  "offense_report_share",
  "defense_report_share"
)

model_df <- model_df |>
  mutate(across(all_of(feature_cols), ~replace_na(.x, 0))) |>
  mutate(
    injury_bucket_num = ntile(injury_severity_score, 3),
    injury_group = case_when(
      injury_bucket_num == 1 ~ "Low injury burden",
      injury_bucket_num == 2 ~ "Medium injury burden",
      TRUE ~ "High injury burden"
    ),
    injury_group = factor(injury_group, levels = c("Low injury burden", "Medium injury burden", "High injury burden"))
  )

model_df |>
  summarise(
    rows = n(),
    min_season = min(season),
    max_season = max(season)
  )

regression_metrics <- function(actual, predicted) {
  tibble(
    rmse = sqrt(mean((actual - predicted)^2)),
    mae = mean(abs(actual - predicted)),
    test_r2 = cor(actual, predicted)^2
  )
}

classification_metrics <- function(actual, predicted, positive = "Yes") {
  actual <- as.character(actual)
  predicted <- as.character(predicted)

  tp <- sum(actual == positive & predicted == positive)
  tn <- sum(actual != positive & predicted != positive)
  fp <- sum(actual != positive & predicted == positive)
  fn <- sum(actual == positive & predicted != positive)

  accuracy <- (tp + tn) / (tp + tn + fp + fn)
  precision <- ifelse(tp + fp == 0, NA_real_, tp / (tp + fp))
  recall <- ifelse(tp + fn == 0, NA_real_, tp / (tp + fn))

  tibble(
    accuracy = accuracy,
    precision = precision,
    recall = recall
  )
}

6. Discussion / Implications

  • Teams with lower injury burden in Weeks 1-8 tend to perform better overall.
  • Injury severity is more useful than simple injury count alone.
  • Injury data helps prediction, but many other factors still matter (QB play, coaching, depth, schedule).

7. Conclusion

This beginner-focused analysis shows a clear pattern:

  • more severe early injuries are generally linked to worse team outcomes,
  • but injury burden is only one piece of performance.

The project includes:

  • clear feature engineering,
  • multiple supervised models,
  • unsupervised clustering,
  • easy-to-read visuals and plain-language interpretation.
