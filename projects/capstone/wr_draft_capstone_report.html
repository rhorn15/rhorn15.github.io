Capstone Report: Forecasting NFL Wide Receiver Production from Draft Information

First Five Seasons, Draft Classes 2005-2015

1 Research Question

How well does draft position predict wide receiver production over the first five NFL seasons, and how much do other pre-draft variables improve that prediction?

To make the comparison fair across players, the report uses wide receivers drafted from 2005 to 2015 and measures production only across each player’s first five seasons.

3 Build the Dataset

The analysis uses:

  1. draft.csv for draft position and player information.
  2. nflreadr::load_player_stats() for season-level receiving production.
  3. combine.csv for pre-draft testing.
  4. passer.csv and nflreadr::load_player_stats() for draft-team passing context.
Code

drafted_wr <- draft |>
  filter(position == "WR", draft >= 2005, draft <= 2015) |>
  transmute(
    playerId = as.character(playerId),
    nameFull,
    draft_year = draft,
    draft_year_centered = draft - 2005,
    round = as.integer(round),
    pick = as.integer(pick),
    log_pick = log(pick),
    draftTeam,
    draftTeamNorm = case_when(
      draftTeam == "ARZ" ~ "ARI",
      draftTeam == "BLT" ~ "BAL",
      draftTeam == "CLV" ~ "CLE",
      draftTeam == "HST" ~ "HOU",
      draftTeam == "SL" ~ "LA",
      draftTeam == "SD" ~ "LAC",
      draftTeam == "OAK" ~ "LV",

    ),
    draftTeamId = teamId,
    ageAtDraft,
    heightInches,
    weight
  ) |>

    by = c("draft_year", "pick")
  )

player_stats_reg <- load_player_stats(2004:2019) |>
  filter(season_type == "REG")

season_receiving <- player_stats_reg |>
  filter(season >= 2005) |>
  group_by(player_id, season) |>
  summarise(
    season_rec_yards = sum(receiving_yards, na.rm = TRUE),
    season_targets = sum(targets, na.rm = TRUE),
    season_receptions = sum(receptions, na.rm = TRUE),
    .groups = "drop"
  )

first_five <- drafted_wr |>
  select(playerId, draft_year, gsis_id) |>
  left_join(season_receiving, by = c("gsis_id" = "player_id")) |>
  filter(is.na(season) | (season >= draft_year & season <= draft_year + 4)) |>
  group_by(playerId, draft_year) |>
  summarise(
    first5_rec_yards = sum(coalesce(season_rec_yards, 0), na.rm = TRUE),
    first5_targets = sum(coalesce(season_targets, 0), na.rm = TRUE),
    first5_receptions = sum(coalesce(season_receptions, 0), na.rm = TRUE),
    peak_season_yards = max(coalesce(season_rec_yards, 0), na.rm = TRUE),
    .groups = "drop"
  )

team_pass_attempts <- passer |>
  inner_join(plays, by = "playId") |>
  inner_join(games, by = "gameId") |>

  group_by(teamId, season) |>

team_rec_yards <- player_stats_reg |>
  filter(!is.na(recent_team)) |>
  group_by(team_code = recent_team, season) |>
  summarise(prev_team_rec_yards = sum(receiving_yards, na.rm = TRUE), .groups = "drop")

combine_data <- combine |>
  transmute(
    playerId = as.character(playerId),
    combine40yd,
    combineVert,
    combineBroad,
    combineBench,
    combineShuttle,
    combine3cone
  )

receiver_epa_by_season <- receiver |>
  mutate(playerId = as.character(playerId)) |>
  inner_join(plays |> select(playId, gameId, evPlay), by = "playId") |>
  inner_join(games, by = "gameId") |>
  filter(seasonType == "REG") |>
  group_by(playerId, season) |>
  summarise(
    season_targets_epa = n(),
    season_total_epa = sum(coalesce(evPlay, 0), na.rm = TRUE),
    .groups = "drop"
  )

first_five_epa <- drafted_wr |>
  select(playerId, draft_year) |>
  left_join(receiver_epa_by_season, by = "playerId") |>
  filter(is.na(season) | (season >= draft_year & season <= draft_year + 4)) |>
  group_by(playerId, draft_year) |>
  summarise(
    first5_targets_epa = sum(coalesce(season_targets_epa, 0), na.rm = TRUE),
    first5_total_epa = sum(coalesce(season_total_epa, 0), na.rm = TRUE),
    first5_epa_per_target = if_else(first5_targets_epa > 0, first5_total_epa / first5_targets_epa, NA_real_),
    .groups = "drop"
  )

analysis_data <- drafted_wr |>
  left_join(first_five, by = c("playerId", "draft_year")) |>
  left_join(first_five_epa, by = c("playerId", "draft_year")) |>
  left_join(combine_data, by = "playerId") |>
  left_join(
    team_pass_attempts |>
      rename(draftTeamId = teamId, draft_year = season) |>
      mutate(draft_year = draft_year + 1),
    by = c("draftTeamId", "draft_year")
  ) |>
  left_join(
    team_rec_yards |>
      rename(draftTeamNorm = team_code, draft_year = season) |>
      mutate(draft_year = draft_year + 1),
    by = c("draftTeamNorm", "draft_year")
  ) |>
  mutate(
    first5_rec_yards = coalesce(first5_rec_yards, 0),
    first5_targets = coalesce(first5_targets, 0),
    first5_receptions = coalesce(first5_receptions, 0),
    peak_season_yards = coalesce(peak_season_yards, 0),
    first5_targets_epa = coalesce(first5_targets_epa, 0),
    first5_total_epa = coalesce(first5_total_epa, 0),
    first5_yards_per_target = if_else(first5_targets > 0, first5_rec_yards / first5_targets, 0),
    first5_epa_per_target = if_else(first5_targets_epa > 0, first5_total_epa / first5_targets_epa, NA_real_),
    log_first5_rec_yards = log1p(pmax(first5_rec_yards, 0)),
    draft_round = factor(round),
    pick_band = factor(
      case_when(
        pick <= 32 ~ "Picks 1-32",
        pick <= 100 ~ "Picks 33-100",
        pick <= 200 ~ "Picks 101-200",
        TRUE ~ "Picks 201+"
      ),
      levels = c("Picks 1-32", "Picks 33-100", "Picks 101-200", "Picks 201+")
    ),
    era = if_else(draft_year <= 2010, "2005-2010", "2011-2015")
  )

The final dataset has 352 drafted wide receivers. It includes 58 players with zero receiving yards and 52 players with zero targets in their first five seasons.

5 Predictive Modeling

The models use a time-ordered split:

  1. Train on draft classes from 2005 to 2012
  2. Test on draft classes from 2013 to 2015

The five models are:

  1. Pick only
  2. Pick plus draft round and draft year trend
  3. A richer pre-draft model with age, size, and combine variables
  4. A full model that also adds draft-team passing context
  5. A random forest version of the pre-draft profile model
Code 
train_data <- analysis_data |>
  filter(draft_year <= 2012)

test_data <- analysis_data |>
  filter(draft_year >= 2013)

impute_columns <- c(
  "ageAtDraft", "heightInches", "weight", "combine40yd",

  "combineShuttle", "combine3cone",

calc_reg_metrics <- function(actual, pred) {
  tibble(
    rmse = sqrt(mean((actual - pred)^2)),
    mae = mean(abs(actual - pred)),
    medae = median(abs(actual - pred)),
    r2 = 1 - sum((actual - pred)^2) / sum((actual - mean(actual))^2)
  )
}

calc_weighted_errors <- function(actual, pred, weights) {
  tibble(
    w_rmse = sqrt(weighted.mean((actual - pred)^2, w = weights)),
    w_mae = weighted.mean(abs(actual - pred), w = weights)
  )
}

model_results <- bind_rows(
  calc_reg_metrics(actual, pred_pick_only) |> mutate(model = "Pick only"),
  calc_reg_metrics(actual, pred_draft_context) |> mutate(model = "Draft context"),
  calc_reg_metrics(actual, pred_pre_draft) |> mutate(model = "Pre-draft profile"),
  calc_reg_metrics(actual, pred_full) |> mutate(model = "Full model"),
  calc_reg_metrics(actual, pred_rf_pre_draft) |> mutate(model = "Random forest (pre-draft)")
) |>
  select(model, rmse, mae, medae, r2)

5.1 Rolling Draft-Year Cross-Validation

To reduce dependence on one holdout split, the report also uses a rolling blocked validation by draft year. Each fold trains on all earlier draft classes and tests on one later class (2010 to 2015).

Code 
rolling_years <- sort(unique(analysis_data$draft_year))
rolling_years <- rolling_years[rolling_years >= 2010]

rolling_cv_results <- lapply(rolling_years, function(test_year_fold) {
  train_fold <- analysis_data |>
    filter(draft_year < test_year_fold)

  test_fold <- analysis_data |>
    filter(draft_year == test_year_fold)

  if (nrow(train_fold) == 0 || nrow(test_fold) == 0) {
    return(NULL)
  }

  fold_medians <- sapply(train_fold[impute_columns], function(x) median(x, na.rm = TRUE))

  for (column in impute_columns) {
    train_fold[[column]][is.na(train_fold[[column]])] <- fold_medians[[column]]
    test_fold[[column]][is.na(test_fold[[column]])] <- fold_medians[[column]]
  }

  fold_pick <- lm(pick_only_formula, data = train_fold)
  fold_context <- lm(draft_context_formula, data = train_fold)
  fold_pre_draft <- lm(pre_draft_formula, data = train_fold)
  fold_full <- lm(full_formula, data = train_fold)

  set.seed(410 + test_year_fold)
  fold_rf <- ranger(
    formula = pre_draft_formula,
    data = train_fold,
    num.trees = 500,
    min.node.size = 5,
    importance = "none"
  )

  fold_actual <- test_fold$first5_rec_yards

  fold_metrics <- bind_rows(
    calc_reg_metrics(fold_actual, pmax(predict(fold_pick, newdata = test_fold), 0)) |> mutate(model = "Pick only"),
    calc_reg_metrics(fold_actual, pmax(predict(fold_context, newdata = test_fold), 0)) |> mutate(model = "Draft context"),
    calc_reg_metrics(fold_actual, pmax(predict(fold_pre_draft, newdata = test_fold), 0)) |> mutate(model = "Pre-draft profile"),
    calc_reg_metrics(fold_actual, pmax(predict(fold_full, newdata = test_fold), 0)) |> mutate(model = "Full model"),
    calc_reg_metrics(fold_actual, pmax(predict(fold_rf, data = test_fold)$predictions, 0)) |> mutate(model = "Random forest (pre-draft)")
  ) |>
    mutate(test_year = test_year_fold, n_test = nrow(test_fold))

  fold_metrics
}) |>
  bind_rows()

rolling_cv_summary <- rolling_cv_results |>

7 Conclusion

Draft position is the strongest and most consistent predictor of early-career receiving production for drafted NFL wide receivers. Earlier picks generally receive more opportunity and produce more receiving yards across their first five seasons, but draft capital does not fully explain future performance.

Adding pre-draft and contextual information improves prediction compared with using pick number alone. The full modeling results show that receiver outcomes are shaped by a combination of draft investment, player profile, opportunity, and uncertainty. The random forest and richer linear models help capture nonlinear relationships, but even the best models still miss extreme steals and busts.

The practical takeaway is that draft position is useful, but it should be treated as a signal rather than a guarantee. Teams can improve forecasting by combining draft capital with additional player characteristics, while still recognizing that wide receiver development includes major uncertainty.

8 Limitations

This project focuses only on drafted wide receivers from 2005 to 2015 and measures production during the first five NFL seasons. That design gives each player a fair evaluation window, but it leaves out undrafted players and more recent draft classes.

The analysis is also limited by the available pre-draft variables. Some important factors, such as route running, injury history, team fit, quarterback quality, coaching, and player development environment, are difficult to measure with the available data. Because those factors affect receiver production, the models should be interpreted as predictive tools rather than complete explanations of NFL success.
